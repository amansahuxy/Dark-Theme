# 🌙 Dark Mode Toggle App

A simple JavaScript project that allows users to switch between **Dark Mode** and **Light Mode** by clicking buttons.

## 🚀 Features

* Switch to Dark Mode
* Switch to Light Mode 
* Instant theme change
* Beginner-friendly project
* Built with pure JavaScript

## 🛠️ Technologies Used

* HTML5
* JavaScript

## 📁 Project Structure

```text
Dark-Mode-App/
│
├── index.html
├── script.js
└── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/dark-mode-app.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## 🧠 How It Works

* The app selects the **Dark** and **Bright** buttons using `getElementById()`.
* When the **Dark** button is clicked, the page background changes to black.
* When the **Bright** button is clicked, the page background changes back to white.
